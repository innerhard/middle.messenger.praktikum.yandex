## Мессенджер Giga-chat
### Описание

Супер мега ультра чат от Яндекс практикум

### Установка проекта
- ```npm run dev```   - запуск в develop режиме
- ```npm run start``` - собираем билд и запускаем сервер
- ```npm run build``` - собираем билд
